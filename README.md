# Global Bottom Navigation Bar

A new Flutter package that implements Global Bottom Navigation Bar Feature like
that one in native development.

[![pub.dev](https://img.shields.io/static/v1?label=pub.dev&message=v0.0.6&color=blue)](https://pub.dev/packages/global_bottom_navigation_bar)

## Getting Started

Did you try to make Global Bottom Navigation bar to your app and you failed many times Did you spend a lot of hours to do this fancy thing but you never succeed. Ok, do not worry, we are here to help and we brought a new library for you simple to use, scalable to develop and light to integrate.
